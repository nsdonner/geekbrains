Название проекта Описание Функционал для реализации Предлагаемые технологии Аналоги
Парсер CDR Asterisk c веб-доступом
АТС на базе ПО Asterisk складывает логи обработаных вызовов в БД, предлагаю написать сервис для удобного их просмотра
  1. Авторизация
  2. Авторизация с телефона
  3. Просмотр звонков за период
  4. Просмотр звонков с фильтрами по номерам/длительности/статусу
  5. Отдельный фильтр по направлениям (внутренние, город/межгород/сотовые/международные)
  6. Предоставление данных в виде диаграмм и графиков, подсчет средней статистики, анализ данных
  7. Вывод данных в виде отчетного документа
  8. Примерная стоимость звонков
  9. Вывод баланса из сторонних источников(баланса у провайдера например)
  10. Возможность коментировать вызовы, и направления/номера(сопоставление с телефонной книжкой)
php, mysql, html/css, js и его фрэймворки по построению различных таблиц и диаграмм
